About Live Code Docs

Thank you for using Live Code Docs! Live Code Docs is an online, collaborative writing tool designed to help multiple developers work on the same piece of code at the same time without complicated merge errors. Please note that this is simply a prototype and therefore not all of the features are complete. Future releases will provide funcionality to the file, project, settings and save buttons on the top bar. They will also provide the ability to delete and rename files and several other capabilities.

Thank you for joining us as we make Live Code Docs the best it can possibly be. If you have any questions, comments or concerns, please do not hesitate to reach out to the company founder via email at cookmn@rose-hulman.edu.

Thanks and happy coding!
